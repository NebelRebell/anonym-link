# Anonyme Links

Dieses Repository enthält eine HTML-Seite, die eine Funktion zum **Anonymisieren von Links** bereitstellt.  
Damit können Benutzer ihre URLs anonymisieren, um ihre Privatsphäre beim Teilen oder Klicken von Links besser zu schützen.

---

## ✨ Funktionen

- **Eingabevalidierung**: Nur gültige URLs werden verarbeitet.  
- **Mehrere Ausgabeformate**: Direktlink, klickbarer Link, HTML-Link, BBCode-Link und Markdown-Link.  
- **Copy-Buttons**: Jeder generierte Link kann mit einem Klick in die Zwischenablage kopiert werden.  
- **Responsive Design**: Optimierte Darstellung für Desktop und Mobilgeräte.  
- **Sicherheit**: Links werden mit `rel="noreferrer noopener"` sowie `meta referrer="no-referrer"` ausgegeben.  
- **Urheberhinweis**: NebelRebell / Nebelbank.net im Footer integriert.  

---

## 🚀 Verwendung

1. Lade die HTML-Datei herunter oder klone dieses Repository.  
2. Öffne die Datei in einem modernen Webbrowser.  
3. Gib die gewünschte URL ein und klicke auf **Anonymisieren**.  
4. Die anonymisierten Links werden in allen verfügbaren Formaten angezeigt.  
5. Kopiere den passenden Link über den jeweiligen Button und füge ihn ein, wo du ihn brauchst.  

> Standardmäßig wird der Dienst [anonym.to](https://anonym.to) verwendet.  
> Für maximale Kontrolle kannst du im Code auch eigene Proxy- oder Anonymisierungsdienste eintragen.  

---

## ⚠️ Hinweis

Dieses Tool anonymisiert Referrer-Daten, **ersetzt aber kein VPN**.  
Für umfassenden Schutz vor Überwachung im Netz wird empfohlen, zusätzlich einen vertrauenswürdigen VPN oder einen eigenen Proxy zu nutzen.
